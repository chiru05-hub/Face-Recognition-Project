Face Recognition using PCA and ANN:

Project Description:
This project implements a face recognition system using Principal Component Analysis (PCA) and Artificial Neural Network (ANN).

Features:
- Face database generation
- Mean face calculation
- PCA (Eigenfaces)
- ANN training
- Face recognition
- Accuracy vs Eigenfaces (k)
- Open-set recognition using imposter images

Technologies:
- Python
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib

Dataset:
Dataset provided by the internship.

:Results
- Face Recognition Accuracy: 75%
- Known Person Accuracy: 63.33%
- Imposter Detection Rate: 24.24%
